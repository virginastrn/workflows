## Deskripsi Proyek
Proyek ini dibuat untuk memenuhi tugas akhir mata kuliah mengenai **Continuous Integration dan Continuous Delivery (CI/CD)**. Repositori ini mendemonstrasikan penggunaan **GitHub Actions** untuk menjalankan otomatisasi pengecekan kode (Linting) dan pengujian unit (Unit Testing).

## Fitur Utama
* **Otomatisasi Alur Kerja (Workflows):** Terintegrasi dengan GitHub Actions.
* **Linting:** Menggunakan `flake8` untuk memastikan kode Python sesuai dengan standar PEP 8.
* **Unit Testing:** Menggunakan `nose` untuk menjalankan pengujian secara otomatis setiap kali ada perubahan kode (*push* atau *pull request*).

## Struktur Repositori
* `.github/workflows/workflow.yml`: Berisi konfigurasi pipeline CI.
* `README.md`: Dokumentasi utama proyek.
